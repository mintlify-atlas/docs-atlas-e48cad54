## Get started in three steps

<Steps>
  <Step title="Install Operator OS">
    Download the binary for your platform or build from source:
    ```bash
    # Build from source (requires Go 1.21+)
    git clone https://github.com/operatoronline/standard-operator.git
    cd standard-operator
    make deps && make build
    ```
  </Step>
  <Step title="Initialize your workspace">
    Run the interactive onboarding wizard to create your configuration:
    ```bash
    operator onboard
    ```
    This creates `~/.operator/config.json` with your chosen LLM and channels.
  </Step>
  <Step title="Start the Gateway">
    Bring your agent online and connect it to your messaging channels:
    ```bash
    operator gateway
    ```
    You can now message your agent through Slack, Telegram, Discord, or any configured channel.
  </Step>
</Steps>
